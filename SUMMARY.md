## üÜò Need Help?

### Common Issues:

**Build errors?**
```bash
npm run build
```
Check error messages, usually syntax issues.

**Styles not working?**
- Check `src/styles/global.css` for errors
- Verify import in `src/components/Layout.astro`

**Pages not updating?**
- Hard refresh: `Cmd+Shift+R` (Mac) or `Ctrl+Shift+R` (Windows)
- Clear cache
- Restart dev server

**Deployment issues?**
- Check Vercel/Netlify build logs
- Verify Node.js version (16+ required)
- Check `package.json` for errors
